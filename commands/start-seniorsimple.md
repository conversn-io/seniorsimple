# 🚀 Start SeniorSimple Development Server

## Quick Start:
```bash
cd "02-Product-Offers/02-Production-Systems/web-applications/02-SeniorSimple-Platform 2/03-SeniorSimple"
npm run dev -- -p 3010
```

## Navigate to:
- **Local**: http://localhost:3010
- **Network**: http://192.168.1.167:3010

## 🕷️ Test Your Changes:
1. Open browser to http://localhost:3010
2. Open browser console (F12)
3. Run `/crawl-test` to test your fixes locally

## 📋 Port Management:
- **SeniorSimple**: Port 3010 (this project)
- **RateRoots**: Port 3000 (separate project)
- **Auto-detection**: Crawler automatically detects port 3010
